추가할 사항들
  1. Movie table에 영화의 감독에 대한 번호를 의미하는 directorNo와 음향 효과를 담당하는 studio name인 SoundStudio를 추가하라. directorNo는 MovieExec table의 certNo로 나타나야 한다. 또한, SoundStudio 역시 Studio table에 나타나야 한다. 즉, 이 두 애트리뷰트에는 foreign key 제약이 존재한다.
  2. StarsIn table에 각 배우가 영화에 출연할 때 계약한 개런티를 나타내는 Gurantee를 추가하라. 그리고, 이 액수는 최소한 10000 이상이 되어야 한다는 제약을 추가하라.
  3. MovieStar table과 MovieExec table에 각각 배우자의 이름을 나타내는 spouseName를 추가하라. MovieExec table의 spouseName은 MovieStar table의 name을 참조하게 하고, 또한 MovieStar table의 spouseName은 MovieExec table의 name을 참조하게 한다. 또한, MovieExec table에는 제작자의 생년월일을 의미하는 birthdate와 제작자의 성별을 나타내는 gender(MovieStar table의 그것과 동일한..)를 추가하라.
  4. Studio table에 영화사에 근무하는 직원의 수를 의미하는 empNo를 추가하라.
  5. 만화 영화에 대한 새로운 table인 Cartoon을 생성하라. 필요한 애트리뷰트는 만화 영화의 제목과 상영 년도를 의미하는 title과 year, 그리고, 만화 영화에 목소리를 담당하는 사람의 이름을 의미하는 Voice등이다. Cartoon는 Movie table의 subclass로서 저장된 (title, year)는 모두 Movie table에 존재해야 한다. 즉, Cartoon (title, year)는 Movie table의 (title, year)에 대해서 foreign key 역할을 한다. 또한, Voice는 배우이며, 물론 하나의 만화 영화에 여러명의 Voice가 나타날 수 있다.
